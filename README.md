# LTC-Sender-Discord-Bot
Hi, my Name is Labib , This is my first repository. This repository is about a Ltc sender discord bot


To use this bot you first need a few things:
1. Your crypto wallet's private key
2. A discord Bot with all intents on and slash permissions given
3. Your discord bot token
4. A blockypher api key which can be obtained for free using blockcypher api and signing up there
5. And lastly A discord account

To get the private key of your wallet click on the three dots and view private keys

or you can use https://privatekeyfinder.io
